## 1. 部件概述

### 1.1. 产品介绍

微型手指减速机是一种微小型一体化90度传动系统，内部集成了空心杯电机、行星减速器、蜗轮蜗杆减速器、传感器以及驱动控制器，具有位置闭环控制功能。

### 1.2. 技术特点

- 驱控一体化设计
- 小体积、高功率密度、高重复定位精度
- **通信与机械接口：**
  - **电气接口：** RS485 接口，微型手指减速器可以配置地址编号（ID），多个不同 ID 的微型手指执行器可以通过串口总线控制，出厂默认 ID 为 1，通信波特率为 3000000bps，1 起始位、8 数据位、1 停止位、无奇偶校验。
  - **机械接口：** 采用螺丝加销固定。
- **宽电压：** 5V～12V 宽电压范围供电，建议供电 12V。
- 具备过压、过流保护。

### 1.3. 产品系列

微型手指减速器有四个系列可供选择：

- **PH 系列：** 传动技术采用环面包络蜗轮蜗杆。
- **PP 系列：** 外形与 PH 系列一致，传动技术采用普通蜗轮蜗杆。
- **X 系列：**
- **MZ 系列：** 固定方式与 PH 系列不一致。
- **MP 系列：** 外形与 MZ 系列一致，传动技术采用普通蜗轮蜗杆。

## 2. RS-485 通信协议概要

### 2.1. 通信基础

| 项目 | 说明 |
|------|------|
| 通信方式 | RS-485 差分半双工串行通信 |
| 电气接口 | A/B 差分对，3.3V逻辑 |
| 数据格式 | 1 位起始位、8 位数据位、1 位停止位，无奇偶校验 |
| 波特率 | 默认 3000000 bps（不可配置） |
| 通信模式 | 主从问答式，总线轮询 |
| 总线拓扑 | 菊花链或手拉手，建议总线两端并接 120Ω 终端电阻 |
| 最大节点数 | 32 个（标准负载）或 254 个（1/8 负载收发器，ID 范围：1~254） |

### 2.2. 指令帧格式

| 字段名 | 长度 | 说明 |
|--------|------|------|
| 帧头 | 2 B | 0x55 0xAA |
| 帧长度 | 1 B | （帧头）+（长度）+ (ID) + (指令) + (控制表) + (数据)+（校验） |
| ID号 | 1 B | 电机地址（1~254） |
| 指令类型 | 1 B | 指令功能码 |
| 控制表索引 | 1 B | 内存表起始地址 |
| 数据段 | N B | 指令参数或数据内容 |
| 校验和 | 1 B | 所有数据和，取值低8位 |

### 2.3. 应答帧格式

| 字段名 | 长度 | 说明 |
|--------|------|------|
| 帧头 | 2 B | 0xAA 0x55 |
| 帧长度 | 1 B | （帧头）+（长度）+ (ID) + (指令) + (控制表) + (数据)+（校验） |
| ID号 | 1 B | 电机地址 |
| 指令类型 | 1 B | 与指令帧一致 |
| 控制表索引 | 1 B | 内存表起始地址 |
| 数据段 | N B | 返回数据（读指令）或保留（写指令） |
| 校验和 | 1 B | 所有数据和，取值低8位 |

## 4. 关键示例指令

假设我们有一个ID为1的无刷电机，目标位置范围为0-2000。以下是一些最常用的指令示例：

### 4.1. 修改电机ID（从1改为2）

说明：向ID=1的电机写入新的ID号（2）到控制表地址2。修改后立即生效，若需断电保存需执行"参数装订"。

**指令帧：** `55 AA 08 01 02 15 02 21`

| 字段 | 值 | 说明 |
|------|-----|------|
| 55 AA | 帧头 | |
| 08 | 长度 | 2(帧头)+1(长度)+1(ID) + 1(指令) + 1(索引) + 1(数据) +1(校验)= 08 |
| 01 | 目标ID | (当前ID=1) |
| 02 | 指令 | (写取控制表数据) |
| 15 | 控制表索引 | (ID号的地址) |
| 02 | 数据段 | (要写入的新ID) |
| 21 | 校验和 | 55+AA+08+01+02+15+02=0x0121 |

### 4.2. 目标位置设置 – 位置模式（无反馈）

说明：让ID=1的电机运动到位置（度）10。

**指令帧：** `55 AA 0F 01 03 31 00 00 54 60 10 00 20 00 27`

| 字段 | 值 | 说明 |
|------|-----|------|
| 55 AA | 帧头 | |
| 0F | 长度 | 2(帧头)+1(长度)+1(ID) + 1(指令) + 1(索引) + 8(数据) +1(校验)= 0F |
| 01 | 目标ID | |
| 03 | 指令 | (指令控制模式) |
| 31 | 控制表索引 | (位置模式（无反馈）) |
| 00 00 54 60 | 角度 | 最高位转向，5460=21600/1080/2=10 |
| 10 00 | 预留 | |
| 20 00 | 力矩 | 2000=8192 |
| 27 | 校验和 | 55+AA+0F+01+03+31+00+00+54+60+10+00+20+00 = 0x0227 |

### 4.3. 紧急停止

说明：立即停止ID=1的电机的所有运动。

**指令帧：** `55 AA 08 01 03 2A 00 35`

| 字段 | 值 | 说明 |
|------|-----|------|
| 55 AA | 帧头 | |
| 08 | 长度 | 2(帧头)+1(长度)+1(ID) + 1(指令) + 1(索引) + 1(数据) +1(校验)= 08 |
| 01 | 目标ID | |
| 03 | 指令 | (指令控制模式) |
| 2A | | (急停) |
| 00 | 数据 | (无) |
| 35 | 校验和 | 55+AA+08+01+03+2A=0x0135 |
